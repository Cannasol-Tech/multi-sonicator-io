Detailed Description

GPIO Hardware Abstraction Layer for ATmega32A.

Author
Cannasol Technologies
Date
2025-09-02
Version
1.0.0

Provides hardware-agnostic GPIO control interface for sonicator interfaces, status indicators, and digital I/O operations. Abstracts ATmega32A port manipulation for consistent cross-platform development.

Pin assignments per config/hardware-config.yaml (SOLE SOURCE OF TRUTH). Supports all sonicator control signals, status LED, and system I/O.
