Detailed Description

PWM Hardware Abstraction Layer for ATmega32A.

Author
Cannasol Technologies
Date
2025-09-02
Version
1.0.0

Provides hardware-agnostic PWM interface for sonicator amplitude control. Uses Timer2 to generate PWM signal on PD7 (pin 21) for shared amplitude control of all 4 sonicators.

PWM signal feeds 1kΩ + 10µF RC filter and LM358N op-amp for 0-10V output. Pin assignments per config/hardware-config.yaml (SOLE SOURCE OF TRUTH).
