Detailed Description

Hardware frequency counter implementation using Pin Change Interrupts.

Author
Cannasol Technologies
Date
2025-01-10
Version
1.0.0

Implements real-time frequency measurement for FREQ_DIV10_# pins using Pin Change Interrupts on Port B. Provides high-precision edge counting with minimal CPU overhead and comprehensive error handling.

Calculate frequency for specified channel.

Parameters
channelChannel ID (0-3 for sonicators 1-4)
Returns
Calculated frequency in Hz, 0 if invalid/no signal
  • Returns immediately if no new measurement ready
  • Calculates frequency from edge count and time window
  • Applies hardware multiplier (×10) to get actual frequency
  • Validates result is within expected range
  • Updates statistics and error counters

◆ frequency_counter_init()

bool frequency_counter_init ( void )

Pin Change Interrupt for Port B (FREQ_DIV10_# pins)

Initialize frequency counting system.

Note
This ISR handles all 4 frequency input pins simultaneously

Optimized for minimal execution time while maintaining accuracy.

  • Captures timestamp immediately for precision
  • Detects edges by comparing current vs previous port state
  • Applies noise filtering to reject spurious edges
  • Updates measurement window completion flags

Get instantaneous frequency (last period measurement)

Parameters
channelChannel ID (0-3)
Returns
Frequency based on last two edges, 0 if unavailable

Provides faster frequency updates by measuring time between consecutive edges rather than counting over full window. Less accurate but more responsive to frequency changes.
