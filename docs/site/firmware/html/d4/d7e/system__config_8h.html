Active Sonicator Summary Registers

System-level summary of currently running sonicators

  • REG_ACTIVE_SONICATOR_COUNT: Number of sonicators actively running (0-4)
  • REG_ACTIVE_SONICATOR_MASK: Bitmask of running units (bit0=S1 .. bit3=S4)
  • Both are Read-Only holding registers and should reflect changes within 100 ms of any start/stop or reset event.

Detailed Description

Multi Sonicator I/O Controller configuration.

Author
Cannasol Technologies
Date
2025-06-27
Version
1.0.0 @framework Arduino Framework (chosen for rapid development and proven libraries)

Configuration constants and compile-time settings for the Multi Sonicator I/O Controller (Model: CAN-SON-CTRL-4). Uses Arduino Framework for UART, I2C, PWM, and ADC operations. See config/hardware-config.yaml — SOLE SOURCE OF TRUTH for hardware pin assignments. This file mirrors the configuration; do not modify mappings here without updating the configuration first.
