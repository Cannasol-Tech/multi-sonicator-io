Detailed Description

MODBUS Register Map Management Interface.

Author
Cannasol Technologies
Date
2025-09-02
Version
1.0.0

Provides high-level interface for managing MODBUS register values and integrating with sonicator control systems. Handles register validation, value limits, and system state synchronization.
