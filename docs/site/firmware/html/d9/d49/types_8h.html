Author
Cannasol Technologies
Date
2025-06-27
Version
1.0.0 @framework Arduino Framework (chosen for rapid development and proven libraries)

Common data structures, enumerations, and type definitions used throughout the Multi Sonicator I/O Controller system. Optimized for Arduino Framework with ATmega32A microcontroller.
