Detailed Description

Sonicator control structure.

This structure contains the control inputs for the sonicator. These are the values that are written to the sonicator by the controller.

Note
The control values are not the same as the status values. The control values are the commands that are sent to the sonicator to change it's state. The status values are the actual state of the sonicator.'
Parameters

safety_overrideSonicator Safety Override (true/false) Read from REG_SON#_CTRL_SAFETY_OVERRIDE
amplitude_controlSonicator Amplitude Control (0-100) Read from REG_SON#_CTRL_AMPLITUDE
Note
The amplitude_control value is not the same as the amplitude_actual value.
