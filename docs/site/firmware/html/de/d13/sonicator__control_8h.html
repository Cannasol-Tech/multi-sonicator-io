Detailed Description

Single Sonicator Control API for Epic 1 Story 1.4.

Note
Developed by Cannasol Technologies @project Multi Sonicator IO Controller
Date
2025-09-12
Version
1.0.0

Provides clean API for single sonicator control with proper state machine, safety requirements, and HAL integration. Designed for Story 1.4: Single Sonicator Control Implementation.

Features:

  • State machine with proper safety transitions
  • Non-blocking update() for 1kHz main loop
  • HAL abstraction for testability
  • MODBUS register integration ready
  • Comprehensive error handling
