Detailed Description

ADC Configuration (per channel)

  • 10kΩ voltage divider for input protection
  • 0-10.88V input range (2000W × 5.44mV/W)
  • 0-5.44V after divider, 0-1024 ADC counts
  • Resolution: ~2 watts per count

Configuration for direct frequency counting on FREQ_DIV10_# pins

Counts rising edges on CT2000 frequency output (÷10) to measure actual frequency

Frequency Measurement Configuration.

  • Input: 1.9-2.1 kHz (CT2000 frequency ÷ 10)
  • Method: Pin Change Interrupt edge counting
  • Measurement Window: 1 second typical
  • Output: Calculated frequency * 10 = actual sonicator frequency
  • Expected Range: 19-21 kHz actual frequency
